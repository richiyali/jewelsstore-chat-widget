## LLM Suite and integrated assistant

The plugin can create a dedicated assistant page such as:

`/assistant-ia/`

This page can have:

- custom title
- custom slug
- intro message
- autonomous rendering mode
- site logo or custom logo
- brand wording
- color system
- halo / glow effect
- quick suggestion prompts

## Public content indexing

The plugin can summarize public content to enrich assistant responses.

Examples of indexable content:

- pages
- posts
- products
- selected public site content types

## Controlled internal connectors

For authorized logged-in users, the assistant can use internal summaries from selected sources such as:

- conversation history
- captured leads
- WooCommerce orders
- WordPress users

Security options can include:

- requiring a logged-in session
- role-based access
- masking sensitive information
- sending only summaries, not raw internal content
- logging internal-context usage

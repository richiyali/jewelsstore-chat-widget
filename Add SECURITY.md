# Contributing to JewelsStore Chat Widget

Thank you for your interest in contributing to JewelsStore Chat Widget.

We welcome constructive contributions, feedback, ideas, and practical suggestions related to the WordPress plugin ecosystem around this project.

## Ways to contribute

You can contribute by:

- reporting bugs
- suggesting new features
- improving documentation
- sharing use cases
- proposing UX improvements
- testing plugin behavior
- suggesting integrations
- participating in Discussions

## Before submitting changes

Please make sure you:

- describe the problem clearly
- explain the expected result
- mention the affected area
- include screenshots or logs when relevant
- indicate whether the issue concerns the free plugin, premium add-on, assistant page, licensing workflow, or related modules

## Feature suggestions

For feature ideas, please use the Discussions area when possible.

Good feature requests usually explain:

- what problem you want to solve
- who the feature helps
- what result is expected
- whether it affects the free version, premium version, or both

## Bug reports

When reporting a bug, please try to include:

- WordPress version
- plugin version
- PHP version
- active theme
- active plugins that may interact with the feature
- exact steps to reproduce the issue
- screenshots or relevant logs

## Security issues

If the issue is security-related, please do not post it publicly first.

Please follow the instructions in `SECURITY.md` for private reporting.

## Contribution style

Please keep contributions:

- clear
- practical
- respectful
- reproducible
- relevant to the project goals

## Project direction

JewelsStore Chat Widget is focused on:

- WordPress AI assistant experiences
- chat widget workflows
- assistant page experiences
- premium upgrade flows
- content indexing
- controlled internal context
- multilingual readiness
- business and WooCommerce use cases
- WordPress-oriented automation

## Discussions first approach

For many ideas, feedback, or proposals, starting with a GitHub Discussion is preferred before larger changes.

This helps keep contributions aligned with the product direction and avoids duplicated work.

## Thank you

Your feedback, ideas, and real-world usage help improve JewelsStore Chat Widget.

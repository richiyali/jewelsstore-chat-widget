# Free vs Premium

JewelsStore Chat Widget is built with a freemium approach.

The free plugin gives users a practical entry point into the product, while the premium add-on extends the experience with more advanced assistant, licensing, context, and business-oriented capabilities.

## Free version

The free version is designed to provide the core experience and make the plugin easy to discover, install, and start using.

Typical free version areas include:

- floating chat widget
- basic assistant configuration
- core user-facing chat behavior
- foundational branding and interface setup
- WordPress-friendly assistant base
- entry point toward premium trial and premium activation

## Premium version

The premium add-on is designed for deeper assistant usage, broader control, and more advanced business workflows.

Typical premium areas include:

- 12-day premium trial flow
- premium license activation
- dedicated assistant page
- autonomous assistant interface
- public content indexing
- controlled internal connectors
- advanced branding and UI options
- monetized remote campaigns and announcements
- extended business and WooCommerce-oriented workflows

## Why both versions exist

The free version helps users quickly understand the core value of the plugin.

The premium version is built for users who need more advanced assistant capabilities, stronger business workflows, deeper control, or broader monetization and service usage.

## Typical upgrade reasons

Users may choose premium when they need:

- a more advanced assistant experience
- a dedicated assistant page
- richer site knowledge usage
- role-aware internal summaries
- more premium workflows
- more business-oriented automation
- stronger service and monetization positioning

## Positioning

The free version is the discovery and onboarding layer.

The premium version is the expansion layer for more advanced usage, smarter assistant experiences, stronger control, and broader WordPress business value.
